# 📸 Responsive Sidebar Gallery

A simple and responsive sidebar navigation interface built using **HTML** and **CSS**. This project was created to practice modern CSS concepts such as positioning, transitions, hover effects, and building interactive UI components without using JavaScript.
The sidebar smoothly slides in and out using the CSS checkbox hack, while the clean layout and photography-themed background provide a modern user experience.

##  Features
- Responsive sidebar navigation
- Smooth slide-in and slide-out animation
- Built entirely with HTML and CSS
- No JavaScript required
- Hover effects on menu items and icons
- Font Awesome icon integration
- Full-screen background image
- Clean and modern user interface


## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Font Awesome

## Concepts Practiced
While building this project, I explored and practiced:

- Semantic HTML
- CSS Positioning
- CSS Transitions
- Hover Effects
- CSS Selectors
- Responsive Layout
- CSS Checkbox Hack
- UI Design Fundamentals


## 🎯 Purpose

This project was built as part of my frontend development learning journey. The main objective was to understand how interactive navigation menus can be created using only HTML and CSS without relying on JavaScript.
It also helped me gain hands-on experience with CSS transitions, positioning, and creating reusable UI components.
